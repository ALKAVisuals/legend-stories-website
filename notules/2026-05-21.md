### 1.1 Skiper UI v3 — Complete overhaul
- **Wat**: Skiper UI was niet compleet genoeg — geen touch support, geen mobiele fallbacks, geen ARIA labels, video player controls niet functioneel. Volledige rewrite uitgevoerd.
- **Tools/modellen**: JavaScript (vanilla), CSS, GSAP, Swiper.js, IntersectionObserver
- **Stappen uitgevoerd**:
  1. skipper.js volledig herschrijven (402 → ~550 regels) met:
     - Touch detectie (`ontouchstart` / `maxTouchPoints`)
     - Hover Expand: tap/click support, keyboard nav (Enter/Space), ARIA tablist/tab roles
     - Sticky Card Stack: CSS-only mobile fallback (geen GSAP op mobiel)
     - Scroll Image Reveal: IntersectionObserver fallback op mobiel
     - Crowd Canvas: tap om mensen toe te voegen, performance limiter (25 mobiel / 50 desktop)
     - Video Player: functionele play/pause, seek bar (klik-scrub), mute toggle, simulatie playback, keyboard shortcuts (Space/Escape)
     - Perspective Carousel: responsive breakpoints, autoplay pause on hover
     - Card Swiper: autoplay pause on hover, ARIA labels
  2. skipper.css volledig herschrijven met:
     - Mobile breakpoints (@768px) voor alle 7 componenten
     - Hover Expand: vertical stack op mobiel
     - Sticky Cards: stacked cards op mobiel
     - Scroll Reveal: geen sticky op mobiel
     - Carousels: kleinere afmetingen op mobiel
     - Video modal: 95% breedte op mobiel
     - Focus-visible outlines in gold
     - `prefers-reduced-motion` support
  3. index.html updates:
     - Video modal controls: functionele classes (skp-video-play, skp-video-seek, skp-video-mute)
     - ARIA labels op alle interactieve elementen
     - Copy-to-clipboard button op CSS code snippet
     - copyCode() functie toegevoegd
     - Viewport meta: viewport-fit=cover, theme-color, apple-mobile-web-app
     - Sidebar: safe-area-inset, momentum scrolling, cubic-bezier transition
     - Nav links: 44px min-height op mobiel
  4. Alle wijzigingen gepusht (commit 6929dd9)
- **Resultaat**: Skiper UI v3 volledig werkend op desktop én mobiel
- **Eindconclusie**: ✅ Succes
